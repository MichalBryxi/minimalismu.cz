# minimalismu.cz

## Installation

Clone this repository and run following commands from the root directory:

```sh
# Install RoR gems
bundle install
# Go to EmberJS sources directory
cd frontend
# Install npm modules
npm install
# Install bower modules
bower install
# Go back up (to run the app)
cd ..
```

## Running

```sh
rails s
```

Then go to: [http://localhost:3000/](http://localhost:3000/)
